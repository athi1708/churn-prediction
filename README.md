Quick Start
1. Place the dataset
churn_project/data/telco.csv
Download from: https://www.kaggle.com/datasets/blastchar/telco-customer-churn
2. Install dependencies
bashcd backend
pip install -r requirements.txt
3. Train the model
bash# From project root
python training/train.py
This will:

Clean and encode the Telco dataset
Engineer 10 new features
Apply SMOTE oversampling
Run XGBoost with 30-iteration RandomizedSearchCV
Calibrate the decision threshold via F1 optimization
Save model artifacts to backend/model/

Expected output:
AUC-ROC:  ~0.845
F1 Score: ~0.62
Optimal threshold: ~0.38–0.44
4. Start the API
bashcd backend
uvicorn main:app --reload --port 8000
API docs: http://127.0.0.1:8000/docs
5. Open the dashboard
Open frontend/index.html in your browser.
